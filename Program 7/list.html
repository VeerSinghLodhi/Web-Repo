Course Curriculum (270 Hours):

  1. Front-End Development (150 Hours)
    • HTML & HTML5 (25 hours)
      • Introduction to HTML structure and elements
      • Creating semantic markup
      • Working with forms, tables, and multimedia
      • Introduction to HTML5 features
    • CSS & CSS3 (30 hours)
      • Understanding CSS syntax and selectors
      • Styling elements with properties and values
      • Layout techniques using floats, positioning, and flexbox
      • Advanced CSS3 features like animations and transitions
    • JavaScript Fundamentals (40 hours)
      • Core JavaScript concepts (variables, data types, operators)
      • Control flow statements (conditionals, loops)
      • Functions and object-oriented programming basics
      • Introduction to the DOM (Document Object Model) and event handling
    • Bootstrap (20 hours)
      • Getting started with Bootstrap framework
      • Using grids and layouts for responsive design
      • Implementing Bootstrap components (buttons, forms, navigation)
    • Single Page Applications (SPAs) (35 hours)
      • Understanding the concept of SPAs
      • Choosing a front-end framework - covered in detail during labs and projects
      • Building SPAs with practical implementation
  2. Back-End Development (80 Hours)
    • PHP Fundamentals (30 hours)
      • Introduction to PHP syntax and basic scripting
      • Working with variables, data types, and operators
      • Control flow statements (conditionals, loops)
      • Functions and arrays
    • MySQL Database (25 hours)
      • Understanding relational databases and SQL concepts
      • Creating and managing databases, tables, and relationships
      • Performing data manipulation using CRUD operations (Create, Read, Update, Delete)
      • Connecting to MySQL from PHP scripts
    • Server-Side Programming with PHP (25 hours)
      • Sending and receiving data with forms
      • Processing user input with PHP scripts
      • Working with sessions and user authentication
      • Introduction to database interaction (queries)
  3. Project and Portfolio Development (40 Hours)
    • Capstone Project (30 hours):
      • Apply your acquired knowledge by building a full-fledged web application that showcases your full-stack capabilities.
      • Portfolio Development (10 hours): Learn how to curate a compelling portfolio website to effectively present your skills to potential employers.

      • You'll be guided by a team of accomplished instructors with a proven track record of success in the web development industry. Their practical insights and mentorship will ensure you gain the necessary skills and confidence to excel in your career.
