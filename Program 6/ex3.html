The JavaScript typeof Operator

The typeof operator returns the type of a variable, object, function or expression.

Primitive Data A primitive data value is a single simple data value with no additional properties and methods. The typeof operator can return one of these primitive types: string number boolean undefined

JavaScript typeof

The typeof operator returns the type of a variable or an expression.

Complex Data The typeof operator can return one of two complex types: function object The typeof operator returns "object" for objects, arrays, and null. The typeof operator does not return "object" for functions.
